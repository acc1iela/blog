# Accio Blog

Astro製の個人ブログ。

## Tech Stack

- [Astro](https://astro.build/) v5
- React 19
- Tailwind CSS
- MDX

## Getting Started

```bash
# インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 構成

```
src/
├── components/   # UIコンポーネント
├── content/      # ブログ記事 (MDX)
├── layouts/      # レイアウト
└── pages/        # ページ
```

## 要件

- Node.js >= 20.0.0
